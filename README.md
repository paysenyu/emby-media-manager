# Emby Media Manager

Emby Media Manager is a tool for managing your media content effectively.

## Getting Started

Welcome to the Emby Media Manager project! This document will help you set up the project for development.

## Installation

Clone this repository and install the requirements:
```
git clone https://github.com/paysenyu/emby-media-manager.git
cd emby-media-manager
pip install -r requirements.txt
```

## Usage

Run the application using the integrated Docker environment.
